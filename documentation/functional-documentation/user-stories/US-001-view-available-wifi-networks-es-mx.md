# US-001: Ver redes WiFi disponibles

## Historia de Usuario

Como usuario de Void Linux,
quiero ver las redes WiFi disponibles,
para poder elegir a cual red conectarme.

## Criterios de Aceptacion

- Dado que el sistema tiene WiFi disponible, cuando abro la aplicacion, entonces puedo ver una lista de redes WiFi disponibles.
- Dado que existen redes disponibles, cuando se muestra la lista, entonces cada red incluye informacion suficiente para identificarla.
- Dado que no existen redes disponibles, cuando se muestra la lista, entonces la aplicacion informa que no se encontraron redes WiFi.
