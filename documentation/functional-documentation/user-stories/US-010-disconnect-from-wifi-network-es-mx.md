# US-010: Desconectarse de una red WiFi

## Historia de Usuario

Como usuario de Void Linux,
quiero desconectarme de la red WiFi actual,
para detener la conexion inalámbrica activa sin cambiar de red.

## Criterios de Aceptacion

- Dado que estoy conectado a una red WiFi, cuando elijo desconectarme, entonces la aplicacion detiene la conexion WiFi actual.
- Dado que la desconexion es exitosa, cuando termina el proceso, entonces el sistema informa que la conexion WiFi ya no esta activa.
- Dado que no hay una conexion WiFi activa, cuando elijo desconectarme, entonces la aplicacion informa que no existe una conexion WiFi activa para desconectar.
