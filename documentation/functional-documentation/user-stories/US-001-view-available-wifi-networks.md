# US-001: View available WiFi networks

## User Story

As a Void Linux user,
I want to view available WiFi networks,
so that I can choose which network to connect to.

## Acceptance Criteria

- Given WiFi is available on the system, when I open the application, then I can see a list of available WiFi networks.
- Given networks are available, when the list is shown, then each network includes enough information to identify it.
- Given no networks are available, when the list is shown, then the application reports that no WiFi networks were found.
