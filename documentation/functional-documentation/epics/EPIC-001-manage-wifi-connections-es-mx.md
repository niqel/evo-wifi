# EPIC-001: Gestionar conexiones WiFi

## Solicitante

Ing. Gustavo Melendez Villarreal

## Objetivo

Como usuario de Void Linux,
quiero una aplicacion minima de terminal para gestionar conexiones WiFi,
para poder conectarme a redes disponibles y tener acceso a internet.

## Historias de Usuario

- US-001: Ver redes WiFi disponibles
- US-002: Seleccionar y conectarse a una red WiFi
- US-003: Manejar red ya conectada
- US-004: Cambiar a otra red WiFi
- US-005: Solicitar contrasena cuando sea necesario
- US-006: Usar contrasena guardada cuando exista
- US-007: Mostrar contrasena de la conexion actual
- US-008: Olvidar contrasena WiFi guardada
- US-009: Ver estado actual de la red
