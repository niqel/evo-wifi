# EPIC-001: Manage WiFi connections

## Requester

Ing. Gustavo Melendez Villarreal

## Goal

As a Void Linux user,
I want a minimal terminal application to manage WiFi connections,
so that I can connect to available networks and access the internet.

## User Stories

- US-001: View available WiFi networks
- US-002: Select and connect to a WiFi network
- US-003: Handle already connected network
- US-004: Switch to another WiFi network
- US-005: Request password when needed
- US-006: Use saved password when available
- US-007: Show current connection password
- US-008: Forget saved WiFi password
